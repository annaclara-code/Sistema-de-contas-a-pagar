# Sistema de Gestão Financeira

Este é um projeto de estudo em Python que simula um sistema de gestão financeira.

## Funcionalidades

- Cadastro de contas a pagar
- Listagem de contas
- Controle de status (Pago / Pendente)
- Atualização do status das contas

## Objetivo

O objetivo do projeto é praticar programação em Python, manipulação de dados e controle de fluxo, além de aprender a usar Git e GitHub.

## Possíveis melhorias futuras

- Interface gráfica
- Sistema de login
- Relatórios financeiros
